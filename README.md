This package is a wrapper bridging [php-pubsub](https://github.com/Superbalist/php-pubsub) into Laravel.

The following adapters are supported:
* Local
* /dev/null
* Redis
* Kafka
* Google Cloud

## Installation

```bash
composer require superbalist/laravel-pubsub
```
